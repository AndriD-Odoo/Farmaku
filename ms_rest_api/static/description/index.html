Create Odoo REST API with appropriate response format and status code

By default, odoo already has a native API and provides a controller to create a REST API. But we can't determine status code and response format ourselves, because odoo has a static format. So with this module we can:
  • Set the appropriate status code
  • Set the appropriate response format
  • Authentication with API key without username and password
